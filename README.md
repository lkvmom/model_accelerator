# 🔬 Согласование пучка в ускорителе

Веб-приложение для расчёта согласующей секции пучка частиц.

## 🚀 Быстрый старт

```bash
# Создание виртуального окружения
py -3.11 -m venv venv

# Включение виртуального оружения
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
uvicorn main:app --reload

# Открыть в браузере
http://127.0.0.1:8000
